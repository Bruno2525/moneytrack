💰 MoneyTrack Dashboard

O MoneyTrack Dashboard é um sistema web de controle financeiro pessoal desenvolvido para ajudar usuários a gerenciar suas finanças de forma simples, visual e eficiente.

A aplicação permite o registro de receitas e despesas, cálculo automático de saldo, filtragem de movimentações e visualização de histórico financeiro em tempo real. Além disso, conta com uma calculadora inteligente que auxilia na tomada de decisão de compras com base na situação financeira atual do usuário.

🚀 Funcionalidades
Cadastro de receitas e despesas
Cálculo automático de saldo (receitas, despesas e total)
Filtro de movimentações por tipo
Histórico completo de transações
Calculadora de viabilidade de compra
Gráficos automáticos para análise financeira
Modo claro/escuro
Menu de configurações personalizáveis
Salvamento automático com LocalStorage
Layout responsivo (desktop e mobile)
🧠 Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage
🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar habilidades em desenvolvimento front-end, incluindo manipulação de DOM, organização de dados, lógica de programação, persistência de informações no navegador e criação de interfaces responsivas.
